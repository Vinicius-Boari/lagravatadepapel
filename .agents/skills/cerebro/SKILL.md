# Skill Cérebro

Esta skill atua como um coordenador central (aglomerado), ativando e orquestrando todas as outras skills disponíveis no projeto.

## Capacidades

- **Orquestração de Skills**: Ao ser acionada, ela identifica todas as outras skills instaladas e as utiliza de forma integrada.
- **Integração de Vídeo**: Utiliza a `video-creator` para gerar conteúdo visual.
- **Criação de Skills**: Utiliza a `skill-creator` para expandir suas próprias funcionalidades ou criar novas ferramentas sob demanda.
- **Visão Holística**: Processa solicitações complexas que exigem a colaboração de múltiplas especialidades (vídeo, código, design, etc.).

## Uso

Sempre que o usuário solicitar uma tarefa complexa ou mencionar "Cérebro", esta skill deve ser o ponto de entrada para delegar tarefas às outras sub-skills.
